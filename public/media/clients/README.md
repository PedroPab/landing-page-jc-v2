# Logos de clientes

Carpeta para los logos de empresas clientes de JC Resortes.

## Formato esperado

- Formato: PNG (fondo transparente recomendado)
- Tamaño sugerido: 300×120 px (horizontal) o hasta 200×200 px (cuadrado)
- Nombre del archivo: debe coincidir con el campo `logo.src` en `src/content/pages/clients.json`

las iniciales de la empresa en un badge de color.
